## üìö What's Inside

This repository is a structured learning path through Python, organized across **3 progressive notebooks**:

### üìì Python Essentials 1 ‚Äî Foundations
> *The building blocks of Python programming*

- Jupyter Notebook overview & workflow
- Variables, Data Types, and Type Casting
- Operators (Arithmetic, Comparison, Logical, Bitwise)
- Conditional Statements (`if`, `elif`, `else`)
- Loops (`for`, `while`) and Loop Control (`break`, `continue`, `pass`)
- Functions ‚Äî defining, calling, arguments, return values
- String Manipulation and Formatting

---

### üìì Python Essentials 2 ‚Äî Intermediate Concepts
> *Going deeper with Python's core capabilities*

- Lists, Tuples, Sets, and Dictionaries
- List Comprehensions & Generator Expressions
- File Handling ‚Äî reading, writing, appending
- Exception Handling (`try`, `except`, `finally`)
- Modules & Packages ‚Äî importing, creating
- Lambda Functions, `map()`, `filter()`, `reduce()`
- Iterators and Generators

---

### üìì Python Essentials 3 ‚Äî Advanced & Applied
> *Real-world Python for data and design*

- **Object-Oriented Programming (OOP)**
  - Classes & Objects
  - Inheritance, Polymorphism, Encapsulation, Abstraction
  - Magic / Dunder Methods
- **NumPy** ‚Äî array operations, broadcasting, indexing
- **Pandas** ‚Äî DataFrames, data cleaning, groupby
- **Data Visualization** ‚Äî Matplotlib, Seaborn
- **Time Series Analysis**
  - DateTime handling with Pandas
  - Resampling, Rolling Windows
  - Trend & Seasonality decomposition

## üöÄ Getting Started

### Prerequisites

Make sure you have the following installed:

```bash
Python 3.x
Jupyter Notebook or JupyterLab
```

### Installation

```bash
# Clone the repository
git clone https://github.com/vinodbavage31/Python-Basics.git

# Navigate into the directory
cd python-essentials

# Install required libraries
pip install numpy pandas matplotlib seaborn jupyter

# Launch Jupyter Notebook
jupyter notebook
```

Then open any of the `.ipynb` files and run the cells in order.
